Version: 0.7
Date: 2026-07-08
Notes: - Admin / student mode: password-protected admin login (🔒 button).
         Students see Start only; admin sees full word bank management.
         Password hashed with SHA-256 (SubtleCrypto), synced via Firestore.
       - Voice selector: choose any system TTS voice per language mode.
         All voices listed (fixes Han Premium detection on iOS).
       - Non-comic Nunito font replaces Chalkboard SE on all headings.
       - Print fix: display-based CSS; iOS Safari opens Blob popup for printing.
       - Review screen shows session date/time in print output.
       - Hint text doubled to 64 px for easier reading.

Version: 0.4
Date: 2026-07-07
Notes: - Smoother pen writing: use PointerEvent.getCoalescedEvents() to capture
         all sub-points Safari merges, and a low-latency (desynchronized) canvas
         context, so Apple Pencil strokes no longer look broken/choppy.
       - Pinyin hint now uses a clean upright font instead of the cartoon display font.
       - Home screen shows the version number (from APP_VERSION const).

Version: 0.3
Date: 2026-07-07
Notes: - Rotation fix: writing is no longer lost when the iPad rotates

       - Pinyin hint: the reveal (eye button) and the review cards now show
